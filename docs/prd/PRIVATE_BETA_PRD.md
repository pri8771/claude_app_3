# Private Beta PRD — Foldlight

**Version:** 1.0.0 | **Date:** 2026-06-27 | **Status:** Draft
**Beta:** Private Alpha/Closed | **Duration:** 2 weeks

## Objectives
1. Validate fold mechanic intuitive without guidance
2. Identify P0/P1 crashes + data corruption
3. Measure Day-1 and Day-7 retention
4. Test IAP sandbox (3 products)
5. Validate performance on real devices
6. Gather qualitative difficulty feedback

## Scope: In
- Core fold mechanic (all 7 tile rules)
- Tutorial levels 1-30
- Infinite procedural mode
- Daily puzzle
- World restoration first island
- Crystal Cave biome
- IAP sandbox: No Ads, Starter Bundle, Hint Pack 10
- Save/load + tutorial

## Scope: Out
- Other 9 biomes, remaining IAP, Challenge Pass, Game Center
